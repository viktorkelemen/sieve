# Sieve

Max for Live MIDI effect with generative transformations (note skipping, probability, phasing, etc.)

## Installation

1. Copy `Sieve.amxd` to your Ableton User Library
2. Drop it on a MIDI track as an effect

## Development

The device consists of:
- `Sieve.amxd` - The Max for Live device
- `midi-processor.js` - JavaScript logic for MIDI transformations
